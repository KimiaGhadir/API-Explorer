# API Explorer

A modern, lightweight **API Explorer / REST Client** built with **React**, **Vite**, **Tailwind CSS**, and **Axios**.  
This app helps you quickly compose API requests, inspect responses, manage multiple tabs, and keep a local history of your requests.

## Features

- Create and manage multiple request tabs
- Send HTTP requests with support for:
  - `GET`
  - `POST`
  - `PUT`
  - `PATCH`
  - `DELETE`
  - and other HTTP methods
- Add custom headers and query parameters
- Send JSON request bodies for non-GET methods
- View API responses and status codes
- Track request history per tab
- Import/export tab or history data
- Light/Dark theme support
- Persistent tabs stored in `localStorage`

## Tech Stack

- **React**
- **Vite**
- **Tailwind CSS**
- **Axios**
- **Lucide React**
- **ESLint**

# API Explorer

یک **API Explorer / REST Client** سبک و مدرن که با **React**، **Vite**، **Tailwind CSS** و **Axios** ساخته شده است.  
این برنامه به شما کمک می‌کند درخواست‌های API را سریع بسازید، پاسخ‌ها را بررسی کنید، چند تب مختلف را مدیریت کنید و تاریخچه درخواست‌ها را به‌صورت محلی نگه دارید.

## امکانات

- ایجاد و مدیریت چند تب برای درخواست‌ها
- ارسال درخواست‌های HTTP با پشتیبانی از:
  - `GET`
  - `POST`
  - `PUT`
  - `PATCH`
  - `DELETE`
  - و سایر متدهای HTTP
- افزودن هدرهای سفارشی و پارامترهای Query
- ارسال بدنه‌ی JSON برای متدهای غیر `GET`
- مشاهده‌ی پاسخ API و کد وضعیت
- ثبت تاریخچه‌ی درخواست‌ها برای هر تب
- قابلیت Import/Export برای داده‌های تب‌ها یا تاریخچه
- پشتیبانی از حالت روشن/تاریک
- ذخیره شدن تب‌ها در `localStorage`

## تکنولوژی‌ها

- **React**
- **Vite**
- **Tailwind CSS**
- **Axios**
- **Lucide React**
- **ESLint**
